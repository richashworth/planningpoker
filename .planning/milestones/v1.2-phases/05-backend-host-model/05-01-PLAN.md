    GREEN phase — implement in SessionManager:

    1. Add field: `private final ConcurrentHashMap<String, String> sessionHosts = new ConcurrentHashMap<>();`

    2. Add method `getHost(String sessionId)`: returns `sessionHosts.get(sessionId)` (returns null if absent).

    3. Add method `setHost(String sessionId, String userName)`: puts `sessionHosts.put(sessionId, userName)`.

    4. Modify `registerUser(String userName, String sessionId)`: After existing `sessionUsers.put(sessionId, userName)` line, add `sessionHosts.putIfAbsent(sessionId, userName)` so the first registered user becomes host.

    5. Modify `removeUser(String userName, String sessionId)`: After existing removal logic, add host auto-promotion:
       ```java
       String currentHost = sessionHosts.get(sessionId);
       if (currentHost != null && currentHost.equalsIgnoreCase(userName)) {
           List<String> remainingUsers = sessionUsers.get(sessionId);
           if (remainingUsers != null && !remainingUsers.isEmpty()) {
               sessionHosts.put(sessionId, remainingUsers.get(0));
           } else {
               sessionHosts.remove(sessionId);
           }
       }
       ```

    6. Add `sessionHosts.clear()` to `clearSessions()`.

    7. Add `sessionHosts.remove(sessionId)` to the eviction loop in `evictIdleSessions()`.
