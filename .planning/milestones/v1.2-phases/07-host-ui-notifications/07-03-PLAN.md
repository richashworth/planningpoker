    7. Kick dispatch (was line 105): change `kickTarget.toLowerCase()` to just `kickTarget` — send the original username to the server.

    The key principle: `name` in the map callback is now the ORIGINAL username. Use `startCase(name)` only inside JSX display elements. Use `name` (unmodified) for all comparisons against `host` and for all dispatch calls.
