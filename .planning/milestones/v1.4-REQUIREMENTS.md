# Requirements: Planning Poker

**Defined:** 2026-04-09
**Core Value:** Teams can run a complete estimation session — pick a scheme, vote on multiple items with labels, review consensus, and export results — all in real time with no signup.

## v1.4 Requirements

Requirements for Code Quality & Tech Debt milestone. Each maps to roadmap phases.

### Concurrency

- [ ] **CONC-01**: `evictIdleSessions()` runs atomically — concurrent requests cannot observe partially-evicted state
- [ ] **CONC-02**: `createSession` ID generation and registration is atomic — no TOCTOU race between contains and add
- [ ] **CONC-03**: Burst messaging reads a consistent snapshot of session state, not live maps after lock release

### UI Robustness

- [ ] **UI-01**: JoinGame and CreateGame forms show loading state and disable submission while a request is in flight
- [ ] **UI-02**: HTTP errors display as MUI Snackbar notifications instead of browser `alert()` dialogs
- [ ] **UI-03**: Failed vote HTTP request reverts both `reducer_vote` (voted flag) and `reducer_results` (optimistic entry)
- [ ] **UI-04**: Lazy-loaded routes show a visible loading indicator instead of blank screen (`Suspense fallback`)

### Dead Code & Legacy

- [ ] **CLN-01**: Dead WebSocket subscription to `/topic/items/${sessionId}` is removed from frontend
- [ ] **CLN-02**: Dead `getSessions()` method is removed from SessionManager
- [ ] **CLN-03**: `redux-promise` middleware replaced and `createStore` migrated to Redux Toolkit's `configureStore`

### Test Coverage

- [ ] **TEST-01**: `useStomp` hook has unit tests covering connect, subscribe, reconnect, and disconnect lifecycle

## Future Requirements

Deferred to future milestone.

- **SEC-01**: CORS wildcard replaced with explicit allowed origins
- **SEC-02**: Session IDs lengthened beyond 8 characters
- **SEC-03**: Rate limiting on session creation and voting
- **A11Y-01**: `aria-live` region for vote→results transition

## Out of Scope

| Feature | Reason |
|---------|--------|
| LOW severity items (unused deps, @Autowired, @RequestMapping style, Lombok) | Cosmetic — not worth a milestone phase |
| Full Redux Toolkit migration (slices, RTK Query) | CLN-03 covers configureStore + middleware only; full migration is separate |
| Database persistence | In-memory state is a deliberate design choice |
