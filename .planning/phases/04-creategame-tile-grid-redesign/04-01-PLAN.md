**Part A: Create SchemeTile.jsx**

Create `planningpoker-web/src/components/SchemeTile.jsx` — a presentational component for a single scheme tile.

Props:
- `scheme` — object from SCHEME_METADATA (has key, icon, name, description)
- `values` — array of sample value strings from SCHEME_VALUES (undefined for custom)
- `selected` — boolean
- `onClick` — callback
- `isCustom` — boolean (true only for custom scheme)
- `customInput` — React node (the TextField for custom values, rendered inside tile when isCustom && selected)

The tile renders as an MUI `Box` (not `Card` — avoid nesting Card inside Card). Structure:

```
[Box - clickable tile container]
  [Box - row: icon + text column + checkmark]
    [Typography - icon, fontSize 1.8rem]
    [Box - column]
      [Typography variant="subtitle2" fontWeight 700 - scheme name]
      [Typography variant="caption" color="text.secondary" - description]
        (description has className="scheme-description" for responsive hiding)
    [CheckCircleIcon - only when selected, color primary.main, positioned right]
  [Box - value chips row, className="scheme-values"]
    (map values to small Chip components, size="small", variant="outlined")
    (for custom: when isCustom && selected, render customInput node instead of chips)
```

Tile container styling:
- `border: '2px solid'`
- `borderColor`: when selected use `'primary.main'`, else `'divider'`
- `borderRadius: 2` (MUI spacing = 16px)
- `p: 2`
- `cursor: 'pointer'`
- `bgcolor`: when selected use `'action.selected'`, else `'transparent'`
- `transition: 'border-color 0.2s, background-color 0.2s'`
- `'&:hover': { borderColor: 'primary.main', bgcolor: 'action.hover' }` (only when not selected)
- `position: 'relative'`

Use `data-testid="scheme-tile-{scheme.key}"` on the container Box for test targeting.
Also give the container `role="option"` and `aria-selected={selected}` for accessibility and so e2e tests can target by role.

The CheckCircleIcon should be from `@mui/icons-material/CheckCircle`. Import it.

Value chips row: `display: 'flex'`, `flexWrap: 'wrap'`, `gap: 0.5`, `mt: 1`. Each chip: MUI `Chip` with `size="small"`, `variant="outlined"`, `label={value}`.

  sx={{
